Precisaremos configurar o console, pois não será usado parte grafica ainda.

Criar módulo:
File > New > New Module > Selecionar Java Or Kotlin Library - Nomear e definir (nome por padrão sempre minúsculo na primeira letra) > Finish.

Disponibilizar execução do projeto para o módulo criado:
(necessita file de execução primeiro)
Run > Run... > Edit Configurations > + > Application - Nomear e editar > Selecionar a file de execução.
